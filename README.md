 Disease Prediction Using Symptoms

 📌 Project Overview
Disease Prediction Using Symptoms is a Machine Learning project that predicts the most likely disease based on a patient's symptoms. The model is trained using historical symptom data and can classify diseases such as Flu, Cold, and Migraine.

This project demonstrates the complete machine learning workflow, including data preprocessing, model training, prediction, and evaluation using Python and Scikit-learn.

 🎯 Objectives
- Predict diseases based on patient symptoms.
- Build a supervised machine learning classification model.
- Perform data preprocessing and feature selection.
- Evaluate model performance using accuracy metrics
 
📂 Dataset
The dataset contains patient symptoms and their corresponding diagnosed diseases.
 Features
- Fever
- Cough
- Fatigue
- Headache
- Other symptom-related attributes (if available)

 #Target
- Disease

🛠 Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Decision Tree Classifier
  
🚀 Project Workflow
1. Import required libraries.
2. Load the dataset.
3. Perform data preprocessing.
4. Handle missing values.
5. Encode categorical variables.
6. Split the dataset into training and testing sets.
7. Train the Decision Tree model.
8. Evaluate model performance.
9. Predict disease for new patient symptoms.
    
📊 Model
- Algorithm: Decision Tree Classifier
- Task: Multi-class Classification

 📈 Results
The trained model successfully predicts the most probable disease based on the symptoms provided by the user. Model performance is evaluated using accuracy score and classification metrics.
